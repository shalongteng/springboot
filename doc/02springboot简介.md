#1、springboot
    什么是springboot
        一个快速的启动器，用来简化新【Spring应用的搭建】以及开发过程。
        spring boot其实就是一个启动spring项目的一个【工具】而已
        Spring Boot 开箱即用，提供各种【默认配置】
        Spring Boot 的核心思想就是【约定大于配置】，多数 Spring Boot 应用只需要很少的 Spring 配置。
#2、优点
    开箱即用，提供各种【默认配置】
    内嵌式容器简化web项目
    没有冗余代码和xml配置的要求
#3、动态生成banner
    在resource下创建一个banner.txt 文件即可
